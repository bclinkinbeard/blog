Who is this for?

This book assumes you know the basics of D3 but are not familiar with SVG. If D3 in 5 Days was your first exposure to D3 this book should be perfect for you. It assumes no knowledge of SVG, but it does assume you understand things like scales and selections.

How is the book structured?

This book is basically divided into 2 parts. The first 4 chapters cover the basics of SVG itself, and the last 6 chapters introduce D3 APIs and techniques for creating SVG charts programmatically.

  1. SVG Elements
  2. Basic Shapes in SVG
  3. Working with Text in SVG
  4. SVG Groups and CSS Transforms
  5. Creating SVG with D3 APIs
  6. A Blueprint for Charts with Axes
  7. SVG Bubble Charts with D3
  8. SVG Line Charts with D3
  9. SVG Area Charts with D3
  10. Make Any SVG Responsive

The book is chock full of code examples, including several CodePen Pens. The book provides ample context and explanation for each one, but if you'd like to explore them on your own they're available as a collection here.
